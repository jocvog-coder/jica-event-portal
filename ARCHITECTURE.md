# JICA イベント情報ポータル - プロジェクト概要

## 📌 プロジェクト説明

このプロジェクトは、JICA海外協力隊の経験者向けに、複数のソースから自動集約したイベント情報を
カレンダー形式で表示するポータルサイトです。

**主な特徴**：
- ✅ 完全無料（GitHub Pages + GitHub Actions）
- ✅ 自動更新（毎日09:00 に自動スクレイピング）
- ✅ 複数ソース対応（JICA公式、JOCA、地域OB・OG会など）
- ✅ カレンダー表示（日付から簡単検索）
- ✅ レスポンシブデザイン（PC・スマホ対応）
- ✅ メンテナンス不要（GitHub Actions が自動処理）

---

## 📦 提供ファイル一覧

### 1. **index.html** （メインのポータルサイト）
   
   **役割**: ユーザーが実際に見るWebサイト
   
   **機能**:
   - カレンダー表示
   - イベント一覧表示
   - カテゴリフィルター
   - レスポンシブレイアウト
   
   **使用方法**: GitHub Pages に配置するだけで動作

---

### 2. **scrape_jica_events.py** （イベント情報取得スクリプト）

   **役割**: 複数のWebサイトからイベント情報を自動抽出
   
   **対応ソース**:
   - JICA 公式サイト
   - JOCA（日本海外協力隊協会）
   - RSS フィード
   - 地域別 OB・OG 会
   
   **実行方法**:
   ```bash
   python scrape_jica_events.py
   ```
   
   **出力**: `events.json` ファイル

---

### 3. **events.json** （イベント情報データベース）

   **役割**: イベント情報を統一された JSON 形式で保存
   
   **構造**:
   ```json
   {
     "last_updated": "更新日時",
     "total_events": "イベント数",
     "events": [
       {
         "id": "一意のID",
         "title": "イベント名",
         "date": "開催日（YYYY-MM-DD）",
         "time": "開催時間",
         "location": "開催場所",
         "description": "説明",
         "category": "ネットワーキング/同窓会/その他",
         "source_url": "出典URL",
         "source_name": "出典名",
         "collected_date": "取得日時"
       }
     ]
   }
   ```
   
   **自動更新**: GitHub Actions により毎日 UTC 00:00 に更新

---

### 4. **scrape-events.yml** （GitHub Actions 設定）

   **役割**: GitHub Actions のワークフロー設定ファイル
   
   **配置場所**: `.github/workflows/scrape-events.yml`
   
   **動作**:
   - 毎日 UTC 00:00 に `scrape_jica_events.py` を実行
   - 取得したデータを `events.json` に保存
   - 自動で Git に commit & push
   
   **スケジュール**: `0 0 * * *` (毎日 00:00 UTC)

---

## 🔄 自動更新フロー

```
時刻: 毎日 UTC 00:00 (日本時間 09:00)
     ↓
GitHub Actions トリガー
     ↓
仮想マシン起動
     ↓
Python 3.10 セットアップ
     ↓
依存パッケージインストール
     ↓
scrape_jica_events.py 実行
  ├─ JICA公式サイト スクレイピング
  ├─ JOCA スクレイピング
  ├─ RSS フィード 解析
  └─ 地域OB・OG会 データ追加
     ↓
events.json 生成
     ↓
Git commit & push
     ↓
GitHub Pages に反映
     ↓
ユーザーがアクセス時に最新情報を表示
```

## 📊 イベント取得元の詳細

### 1. JICA 公式サイト
- **URL**: https://www.jica.go.jp/
- **方法**: HTML スクレイピング + RSS
- **頻度**: 毎日

### 2. JOCA（日本海外協力隊協会）
- **URL**: https://www.joca.or.jp/
- **方法**: HTML スクレイピング
- **頻度**: 毎日

### 3. 地域別 OB・OG 会
- **取得元**: Facebook、各支部ホームページ
- **方法**: サイトスクレイピング
- **頻度**: 毎日

### 4. RSS フィード
- **取得元**: JICA公式 RSS
- **方法**: feedparser で解析
- **頻度**: 毎日
